# 実装計画: `exportCsv` の `PrintWriter` 化

## 目的
`generateJavaService` 関数内の `exportCsv` メソッドにおいて、CSV出力ロジックを `BufferedWriter` から `PrintWriter` を使用するように改善する。これにより、より柔軟な書式設定と簡潔なコードを実現する。

## 計画

1.  **`index.html` の読み込み**: `index.html` ファイルの内容を読み込む。
2.  **`generateJavaService` 関数の特定**: `generateJavaService` 関数内の `exportCsv` メソッドの箇所を特定する。
3.  **`exportCsv` メソッドの変更**:
    *   `BufferedWriter writer = Files.newBufferedWriter(file.toPath())` の行を、`PrintWriter printWriter = new PrintWriter(Files.newBufferedWriter(file.toPath()))` に変更する。
    *   `writer.write(csvHeader);` を `printWriter.println(csvHeader);` に変更する。
    *   `writer.newLine();` を削除する（`println` が改行を含むため）。
    *   `writer.write(csvRow);` を `printWriter.println(csvRow);` に変更する。
    *   `try (BufferedWriter writer = Files.newBufferedWriter(file.toPath()))` を `try (PrintWriter printWriter = new PrintWriter(Files.newBufferedWriter(file.toPath())))` に変更し、`try-with-resources` で `PrintWriter` が適切に閉じられるようにする。
4.  **インポートの確認**: `java.io.PrintWriter` が適切にインポートされていることを確認し、必要であれば追加する。
